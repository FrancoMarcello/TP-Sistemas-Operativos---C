# TP Sistemas Operativos – Sistema Distribuido en C

Proyecto académico desarrollado en C como parte de la materia Sistemas Operativos (UTN).
Consiste en un sistema distribuido con arquitectura modular, donde los distintos componentes
se comunican mediante sockets y manejan concurrencia utilizando hilos, mutex y semáforos.

Este proyecto me permitió profundizar en programación concurrente, comunicación entre procesos,
debugging y resolución de problemas de sincronización.

## Tecnologías y conceptos utilizados
- Lenguaje C
- Programación concurrente (threads, mutex, semáforos)
- Comunicación entre procesos mediante sockets
- Arquitectura modular
- Makefiles
- Linux

## Estructura general
- master
- worker
- storage
- query_control
- utils

> Proyecto desarrollado con fines académicos.
